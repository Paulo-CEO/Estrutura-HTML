Inserindo links

Vamos criar um link para que nosso usúario conheça o SENAI. O comando para criarmos um link é a TAG a. Se definirmos a target como:
_blank: abre a página em uma nova janela/aba;
_self: abre a página na mesma janela;
_parent: abre a página na mesma janela do link;

Tabelas

A tabela é uma estrutura composta de linhas e colunas, com inúmeras possibilidades de formatação. Para construir uma tabela usamos a TAG table, para construir uma linha usamos a TAG tr, e para criar uma coluna usamos a TAG td.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação e o layout de documentos HTML. Ele permite personalizar elementos como fontes, cores, espaçamento, bordas, alinhamento, entre outros. Com CSS, é possível estilizar não apenas o conteúdo HTML, mas também elementos como tabelas, listas, parágrafos e até mesmo linhas e bordas. Como exemplo ficam as estilizações que realizamos nesta página.
Para tanto precisamos criar um novo documento na pasta do nosso projeto, que deverá conter a extenção .css. JavaScript(JS)

JavaScript (JS) é uma linguagem de programação utilizada principalmente para adicionar interatividade e dinamismo a páginas web. Com o JS, é possível criar efeitos visuais, validar formulários, manipular elementos da página, criar animações e até mesmo realizar requisições assíncronas para atualizar conteúdo sem recarregar a página.
Para tanto precisamos criar um novo documento na pasta do nosso projeto, que deverá conter a extenção .js. Como este é um arquivo diferente do html precisamos realizar a conexão entre eles assim na tag head precisamos conectar por meio da tag script, como na figura abaixo, sendo o main.js o nome dado seguido da extensão do arquivo.
